# Java OOP and Simulations

This repository contains various Java projects, exercises, and simulations designed to strengthen object-oriented programming (OOP) skills and understanding of common simulation scenarios. 

## Project Structure

The repository is organized by lab and self-study exercises:

- **Lab1**: Employee Management System
- **Lab2**: Library Book Management System
- **Lab3**: XoX Game
- **Lab4**: Dungeon Adventure
- **Lab5**: Online Store
- **Lab6**: Car Park Management
- **Lab7**: Shape Calculations
- **Lab8**: Weather Data System
- **Study1**: 2D/Path Geometry
- **Study2**: Missing Word Puzzle
- **Study3**: Tents Puzzle
- **Study4**: Chess Board Simulation
- **Study5**: Elevator System Simulation
- **Study6**: File Processing System

## Features

- Object-Oriented Programming with **inheritance, abstraction, interfaces, and encapsulation**
- Simulations including **Elevator System, Chess, and Car Park**
- Data processing using **file I/O and pipelines**
- Interactive games and puzzles to strengthen logic and problem-solving
